Enum for strategies to handle full log queues. 


\begin{DoxyItemize}
\item This enum defines the strategies that can be used when the log queue is full.
\item It provides options for blocking until space is available or dropping the oldest log entry. 
\begin{DoxyParams}{Parameters}
{\em Block} & Blocked waiting for space in the queue.\\
\hline
\end{DoxyParams}

\item When the queue is full, the logging operation will block until space becomes available. 
\begin{DoxyParams}{Parameters}
{\em Drop\+Oldest} & Drop the oldest log entry when the queue is full.\\
\hline
\end{DoxyParams}

\item When the queue is full, the oldest log entry will be removed to make space for the new log entry.
\item This strategy allows for continuous logging without blocking, but may result in loss of older log entries.
