\doxysubsection{Detailed Description}
A lock-\/free log writing implementation. 


\begin{DoxyItemize}
\item This class provides a thread-\/safe way to write logs to a file using a lock-\/free queue.
\item It allows multiple threads to write logs concurrently without blocking each other.
\end{DoxyItemize}

The class uses a lock-\/free queue to store log messages and a separate thread to write them to a file.
