\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class Lock\+Free\+Queue$<$ T $>$}
A lock-\/free queue implementation using atomic operations. 


\begin{DoxyItemize}
\item This queue is designed to be used in a multi-\/threaded environment where multiple threads can push and pop elements concurrently without locks. 
\begin{DoxyParams}{Parameters}
{\em T} & The type of elements stored in the queue\\
\hline
\end{DoxyParams}
The queue uses a fixed-\/size array of nodes, each containing an atomic tail and head index to manage the queue state.
\item The queue supports push and pop operations, where push adds an element to the end of the queue and pop removes an element from the front.
\item The queue is designed to be lock-\/free, meaning that it does not use mutexes or other locking mechanisms to ensure thread safety.
\item The queue is implemented using a circular buffer, where the capacity is a power of two to optimize index calculations. 
\end{DoxyItemize}
